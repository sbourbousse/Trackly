# Trackly Driver - Application PWA Livreur

Application Progressive Web App (PWA) pour les livreurs Trackly.

## Fonctionnalités

- ✅ Connexion livreur
- ✅ Liste des livraisons du jour
- ✅ Détail d'une livraison
- ✅ Géolocalisation en temps réel
- ✅ Envoi position GPS via SignalR
- ✅ Validation de livraison
- ✅ Interface optimisée usage extérieur (boutons larges, contrastes élevés)

## Développement

```bash
npm install
npm run dev
```

L'application sera accessible sur `http://localhost:5175`

## Installation PWA

1. Ouvrir l'application dans Chrome/Edge mobile
2. Menu → "Ajouter à l'écran d'accueil"
3. L'application s'ouvrira comme une app native

## Configuration

Copiez `.env.example` vers `.env` et configurez les URLs du backend.
