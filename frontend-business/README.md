# Trackly - Frontend Business

Tableau de bord SvelteKit pour la gestion des tournees, livraisons et suivi.

## Demarrage local

```sh
npm install
npm run dev -- --open
```

## Scripts utiles

```sh
npm run check
npm run build
npm run preview
```

## Notes

- UI en Svelte 5 (Runes).
- Les donnees sont mockees pour le moment.
